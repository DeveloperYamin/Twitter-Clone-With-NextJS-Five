## Step: 1 (required)

Run this clone repo command:

```sh
git clone https://github.com/DeveloperYamin/Twitter-Clone-With-NextJS-Five.git
```

## Step: 2 (required)

Install the package and run the app:

```sh
npm install
npm run build
npm run start
```
